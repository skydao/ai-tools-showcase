# AI Tools Showcase 网站需求文档

## 一、项目概述

**项目名称**：AI Tools Showcase
**项目目标**：创建一个专业的 AI 工具导航和展示平台，帮助用户发现和探索最新、最实用的 AI 工具
**项目定位**：AI 工具聚合平台，提供工具介绍、分类浏览、搜索等功能

## 二、核心功能

### 2.1 首页功能
- **精选工具展示**：展示 5-6 个热门/精选的 AI 工具
- **分类导航**：快速访问不同类别的 AI 工具
- **搜索功能**：提供搜索框，快速查找工具
- **工具统计**：展示网站收录的工具总数

### 2.2 工具列表页功能
- **分类筛选**：按类别浏览工具（聊天助手、图像生成、编程辅助等）
- **标签筛选**：按标签筛选工具（对话、写作、编程等）
- **搜索**：通过工具名称或描述搜索
- **排序**：按评分、名称、推荐度排序
- **工具卡片展示**：每个工具显示名称、描述、标签、评分、价格

### 2.3 工具详情页功能
- **详细信息**：工具名称、英文名、详细描述
- **基本信息**：类别、标签、评分、定价模式
- **官方网站链接**：直接跳转到工具官网
- **相关工具推荐**：推荐同类别的其他工具

## 三、页面结构

### 3.1 首页 (/)
- 顶部导航栏（Logo + 菜单）
- Hero 区域（标题 + 简介 + 搜索框）
- 精选工具网格
- 分类快捷入口
- 页脚

### 3.2 工具列表页 (/tools)
- 搜索框
- 分类筛选器（侧边栏或顶部）
- 工具卡片网格
- 分页或无限滚动

### 3.3 工具详情页 (/tools/[id])
- 工具标题和描述
- 工具信息卡片
- 官方网站按钮
- 相关工具推荐

## 四、设计风格

### 4.1 整体风格
- **现代科技感**：简洁、专业、有科技感
- **暗色主题**：深色背景，适合科技类网站
- **卡片式设计**：工具以卡片形式展示，信息清晰
- **渐变色点缀**：使用渐变色作为视觉点缀

### 4.2 配色方案
- **主色调**：深色背景（#0a0a0a 或 #1a1a2e）
- **文字颜色**：浅灰/白色（#ededed）
- **强调色**：渐变紫色/蓝色（#8b5cf6 → #3b82f6）
- **卡片背景**：稍浅的深色（#1f2937）

### 4.3 视觉元素
- 圆角设计
- 阴影效果
- 图标和标签
- 悬停动画

## 五、技术栈

### 5.1 前端框架
- **Next.js 14**：React 框架，支持 App Router
- **React 18**：UI 库
- **TypeScript**：类型安全的 JavaScript
- **Tailwind CSS**：实用优先的 CSS 框架

### 5.2 工具库
- **Lucide React**：图标库
- **framer-motion**：动画库（可选）

### 5.3 数据存储
- **静态数据**：工具数据存储在 TypeScript 文件中
- **未来扩展**：可接入数据库（Prisma + PostgreSQL）

## 六、数据结构

### 6.1 AI 工具数据模型

```typescript
interface AITool {
  id: string;           // 工具唯一标识
  name: string;         // 工具中文名
  nameEn: string;        // 工具英文名
  description: string;   // 工具描述
  category: string;     // 类别
  tags: string[];       // 标签数组
  website: string;      // 官网链接
  pricing: string;      // 定价模式（免费/付费/免费+付费）
  rating: number;       // 评分（0-5）
  featured: boolean;    // 是否精选
}
```

### 6.2 初始数据
收录至少 10 个主流 AI 工具，涵盖：
- 聊天助手：ChatGPT、Claude
- 图像生成：Midjourney、DALL-E、Stable Diffusion
- 编程辅助：GitHub Copilot
- 办公效率：Notion AI、Grammarly
- 搜索工具：Perplexity AI
- 视频创作：Runway
- 开发平台：Hugging Face
- 自动化：Zapier AI

## 七、响应式设计

### 7.1 桌面端 (>1024px)
- 网格布局：3-4 列工具卡片
- 侧边栏筛选器
- 完整的导航栏

### 7.2 平板端 (768px-1024px)
- 网格布局：2-3 列工具卡片
- 顶部筛选器
- 简化的导航栏

### 7.3 移动端 (<768px)
- 网格布局：1 列工具卡片

- 汉堡菜单导航
